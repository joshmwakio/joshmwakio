### Bunifu Accounts ERP and Business Intelligence
Comprehensive management solution for schools featuring accounting software and payroll management system. Built with microservices architecture using C# and ASP.NET Core, with WinForms/WPF front-end and mobile app access.

**Key Features:**
- Accounting and payroll management modules
- SMS notification integration
- Secure data access via Cloudflare Tunneling
- Android mobile app for on-the-go access
- M-PESA payment integration
- Advanced financial reporting system

### B2B Integrated Collections APIs
Payment processing system for Co-operative Bank facilitating seamless transactions between merchants and PioneerFSA aggregation system.

**Key Features:**
- Real-time settlement system
- Payee validation against ERP
- Merchant aggregation system
- B2B, B2C, and C2B transaction support
- Credit scoring algorithms for merchant loans

### Cyber File Tracker
Sophisticated program ensuring the integrity and security of confidential files with advanced tracking mechanisms.

**Key Features:**
- Intuitive UI for file tracking
- Real-time alerts and audit logging
- Global file location monitoring
- Prevention of unauthorized modifications
- User-centric design for streamlined file management
